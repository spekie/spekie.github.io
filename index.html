Repositories

membench is a memory benchmark for evaluating the memory system which outputs different array sizes, and their corresponding different stride value.

cpubench is a processor benchmark that evaluates integer operations, floating point operations, memory access, cache performance and multithreading.

calcbench is another processor benchmark but it uses a simpler approach by evaluating prime number calculations with a minimal algorithm.
